Về Ứng Dụng Này

Đây là một ứng dụng web đơn giản được xây dựng bằng Django framework.

Công Nghệ Sử Dụng:

  • Django 5.0 - Python web framework
  • Python 3.11 - Ngôn ngữ lập trình
  • SQLite - Database mặc định
  • HTML/CSS - Giao diện người dùng

Tính Năng Chính:

  • Hệ thống routing URL linh hoạt
  • Template engine mạnh mẽ
  • Admin interface tích hợp sẵn
  • ORM (Object-Relational Mapping)
